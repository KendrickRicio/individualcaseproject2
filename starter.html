Game Packages

To play Star Citizen, you will need a Game Package.
A Game Package includes both a ship and a digital copy of the game.



The Pledge Store has many options to choose from.



  • If you to spend the least money, consider:

    • Aurora MR ($45)
      • Includes small interior
      • Includes bed
      • More stowage storage
      • Equipped with missiles

    • Mustang Alpha ($45)
      • Better fit for ship combat
      • Better flight performance
      • More cargo storage

  • Otherwise, consider the following ships:

    • Anvil C8X Pisces Expedition ($60)
      • Snub Pathfinder

    • Drake Cutter ($60)
      • Light Freight

    • Origin 100i ($65)
      • Touring

    • Avenger Titan ($70)
      • Light Freight

    • Anvil Arrow ($95)
